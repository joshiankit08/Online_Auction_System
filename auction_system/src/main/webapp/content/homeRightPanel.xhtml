Welcome to the World of Online Auction System Application

This application is created as a course project at Concordia University, Montreal.

The application lets you make a bid for all the active auctions as well as allows you to offer an item for the auction.
All this can be done by simply registering yourself to our 'Online Auction System Application'.

You should see the Navigation bar to the left where you can register as user or sign in and access all the auction resources.
